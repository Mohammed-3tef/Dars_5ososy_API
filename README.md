# Dars 5ososy API

A production-ready RESTful API that connects students with private tutors. Tutors can manage profiles, teaching schedules, and lesson bookings, while students can discover tutors, reserve sessions, leave reviews, and track their learning activities.

## Features
- JWT Authentication & Authorization
- Tutor & Student Profiles
- Subject Management
- Tutor Availability Scheduling
- Lesson Booking System
- Smart Search & Filtering
- Reviews & Ratings
- Favorites
- Notifications
- File Uploads
- Payment Integration (Optional)
- Admin Dashboard Support
